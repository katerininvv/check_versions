# История версий

## 2.0
- Проверка обновлений **3 раза в день** (утро/день/вечер, времена настраиваются)
  вместо еженедельной воскресной рассылки.
- Уведомление приходит **только если что-то обновилось** (иначе — тишина).
  Кнопка «Проверить сейчас» присылает результат всегда.
- Поддержка **любых реестров** образов: Docker Hub, GHCR (ghcr.io), Quay, lscr.io и
  другие OCI-совместимые (авторизация определяется автоматически).
- **Переключатель темы**: авто (по системе) / светлая / тёмная / по времени суток.
- **Фиксированное имя тома данных** (`update_tracker_data`) — данные сохраняются
  при любом способе запуска и переживают любые обновления.
- Бесплатный перевод по умолчанию (Google, без ключа); провайдер выбирается в UI
  (Google / LibreTranslate / MyMemory / DeepL).
- Готовность к авто-деплою через Arcane GitOps.
- Упрощены формы настроек.

## 1.0
- Первый рабочий релиз: отслеживание образов Docker Hub, веб-дашборд с авто-входом
  (обратный прокси), Telegram-уведомления, перевод описаний, самоподписанный TLS,
  упаковка в Docker.
